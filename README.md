# Permo API
1. Clonar proyecto
2. ```yarn install```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Setear las variables de entorno
5. levantar la base de datos
```
docker-compose up
```
6. levantar ```yarn start:dev```

7. ejecutar seed ``` localhost:3000/api/seed```
